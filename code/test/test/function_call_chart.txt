1. 사용자 선택
    - user_choice.get_user_choice(): 사용자가 [가위, 바위, 보] 중 하나를 선택
2. 컴퓨터 선택
    - computer_choice.get_computer_choice(): 컴퓨터가 [가위, 바위, 보] 중 랜덤으로 하나를 선택
3. 승패 결정
    - determine_winner.get_winner(user_choice, computer_choice): 사용자와 컴퓨터의 선택을 비교하여 승패를 결정
4. 결과 출력
    - display_result.show_result(winner): 승패 결과를 화면에 출력
5. 다시하기
    - restart_game.restart(): 다시하기 버튼을 누르면 게임을 다시 시작
